# Prerequisites
- Node.js >= 20
- npm >= 9
- Backend running on `http://localhost:8000` (see X_Backend repo)

# Clone the repository
git clone https://github.com/Salacho96/twitter-clone-frontend.git
cd twitter-clone-frontend

# Install dependencies
npm install

# Run on dev mode
npm run dev

# Check in browser
http://localhost:5173
